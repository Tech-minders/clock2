# React Clock
A sleek, responsive clock application built with React, Vite, and date-fns. 
This project demonstrates clean UI design using Tailwind CSS and precise date formatting.


<img width="628" height="455" alt="image" src="https://github.com/user-attachments/assets/78a49dc3-7178-4c06-bfd9-61a647661c40" />


# Features
- Real-time Formatting: Utilizes date-fns to display dates in a human-readable format 
- Tailwind Styling: Implements custom spacing, shadows, and a blue-themed color palette.
- Responsive: Centers perfectly on the screen for various device widths.

# Tech Stack
Framework: React
Utility Library: date-fns 
Styling: Tailwind CSS
Build Tool: Vite

# Installation & Setup
- Clone the repository:
```
bash
git clone git@github.com:Tech-minders/color-clock.git
cd color-clock
```
- Install dependencies:
```
bash
npm install
```

- Run the development server:
```
bash
npm run dev
```
